# **App Name**: GigGuard AI

## Core Features:

- Secure Authentication & Access Control: Implement user signup/login with role-based access for 'Worker' and 'Admin', directing them to respective dashboards upon authentication.
- AI Income DNA Profiling: Analyze worker timestamps, hours worked, earnings per hour, location, and day to generate detailed income profiles (e.g., peak earning hours, high activity days, expected weekly earnings) using an AI-powered analysis tool.
- Predictive Disruption Detection Engine: Utilize simulated worker GPS data to detect collective disruptions (e.g., if 10 workers go offline in a 2km radius within 5 minutes) and log disruption events with location and worker count using an AI tool to identify critical patterns.
- Dynamic Insurance Policy Management: Enable users to choose between Basic, Pro, and Elite Shield plans. Manage weekly auto-deductions, track policy week progression (X of 4), display next renewal details and amount, and enforce the 4-week minimum commitment for cancellation.
- Personalized Worker Dashboard: Display individual policy status, income DNA insights (charts for peak hours, best days, earnings by window), real-time disruption alerts, tailored plan recommendations, and a direct link to the interactive heatmap.
- Comprehensive Admin Dashboard: Provide system administrators with an overview of total workers, active policies, detailed disruption event logs, and worker analytics for monitoring and management.
- Interactive Disruption Heatmap: Visualize geographical disruption events on an interactive India map (Leaflet or Mapbox), employing a color-coded heat representation (Green, Yellow, Red) and informative tooltips detailing disruption level, estimated earning impact, and practical suggestions.

## Style Guidelines:

- Color Palette Rationale: The concepts of 'Protection', 'Security', and 'AI Technology' inform a sophisticated and trustworthy aesthetic. A dark color scheme is chosen to convey depth and a sense of calm reliability.
- Primary Color: A composed and modern blue (#61A0E6) to symbolize trust, technology, and stability, offering good contrast against a dark background.
- Background Color: A deep, subtly bluish-gray (#1A1E24) serving as a professional and non-distracting canvas, providing a sense of depth inherent to a dark scheme.
- Accent Color: A vibrant cyan (#26B3D9) is used for call-to-actions, alerts, and highlighting key information, providing a striking contrast that catches attention while complementing the primary blue.
- Headline Font: 'Space Grotesk' (sans-serif) for its modern, techy, and bold appearance, perfect for titles and dashboard metrics.
- Body Font: 'Inter' (sans-serif) to ensure excellent readability for detailed policy information, analytics, and all explanatory text, providing a neutral yet clear aesthetic.
- User Interface Layout: A modular, card-based layout featuring clean typography, distinct sections for policy details, income insights, and alerts. A prominent sidebar or top navigation will ensure intuitive access to dashboards and other features, with responsiveness across devices.
- Iconography Style: Use modern, flat, and legible icons that clearly represent their functions within the app, enhancing usability without visual clutter.
- Animations: Subtle transition animations will be incorporated for state changes, data updates, and navigation between views, ensuring a smooth and engaging user experience.
